《冰与铁》—— HP 驱动的动态战斗音频系统

Wwise × Unity

基于 HP 与伤害参数构建 RTPC 驱动的战斗音频系统，实现命中反馈与音乐情绪的动态演变。

System Implementation
  • RTPC (HP)→垂直分层音乐 (Vertical Layering) 与侧链避让（Wwise Meter）：随血量阈值激活 Drone 与 Lead，并实现两者间的动态避让。
  • RTPC (Damage)→多阶命中反馈 (Blend Container)：命中声音从 Normal 到 Broken、Final Blow 的质感演变。
  • State (Game State)→全局状态切换：基于胜负分晓状态触发全局静默。
Design Goal

让音频实时响应数值变化，强化玩家对战斗强度及对峙状态（危险或胜利在望）的感知。

02

基于 RTPC 的环境与脚步交互音频系统

Wwise × Unity

构建基于地面材质与空间距离的交互音频系统，实现脚步声与环境音的动态变化。

System Implementation
  • Switch Group→物理材质反馈：配置四种地面材质（Wood / Grass / Water / Stone），每种 5 个随机变体，消除重复感。
  • RTPC (Distance)→空间边界预警：以玩家与区域边界距离实时映射，通过脚步声 Pitch 的线性偏移提供直观的距离听觉反馈。
  • Spatial Ambience→差异化声学环境：按区域材质切换环境底噪 (Bed)，以鸟鸣、风声、地矿回响、水声强化声学指纹。
Design Goal

通过音频反馈强化玩家对空间与物理边界的感知，利用脚步声音高变化提供直观感性的距离提示，提升交互的功能性。

Sword Combat SFX · Wwise × Unity

五组独立挥击动作的音效设计与引擎集成，基于 Animation Event 驱动 Wwise 精确触发。

模型来源：FreeSword Animations · EEJANAI Team · Unity Asset Store (Free)

Design Focus

混合合成 ——《鸣潮》绯雪技能音效重设计

Hybrid Synthesis · In-The-Box

聚焦全板内 (In-The-Box) 混合合成声音重设计。核心材质（冰雪、铃铛）选用精选预设，其余元素（Whoosh, Ice Crack, Pulse Wave, Burst, Eject）经 Ableton Live 内置工具与外部合成器自制调制，部分融合预设低音增强冲击力。通过声音叠加、多参数自动化 (Automation) 与滤波扫频 (Filter Sweep)，塑造冰系技能在蓄力、弹射、爆发阶段的声压感与物理冲击力。

Toolchain

Ableton Live (Built-in Devices) · Vital · Surge XT · OB-Xd  |  视频画面源自 IGN 官方演示片段

Design Focus
  • 混合合成材质→Operator / Wavetable / Vital / Surge XT 自制合成与预置相融合，构建质感统一、结晶感强烈的冰雪攻击特征。
  • 空间与动态平衡→以 Glue Compressor、Saturator、Reverb 粘合元素层，统一空间感；提升子编组饱和度，在提升整体响度时不掩蔽高频细节。

中世纪冷兵器战斧音效设计

Medieval Battle Axe · Ref. For Honor

以《For Honor》音效审美为基准，解构战斧音效音频层级，设计冷兵器物理动态。

Design Focus
  • 材质一致性→共享金属基底与谐音层确保不同动作间音色连贯；共享混响层统一空间感。
  • 受击反馈层次→多层素材叠加，构建重金属兵器撞击的多样打击感。
  • 动态与动能模拟→以自动化控制与包络修剪模拟战斧挥动的动能变化与速度感。

动作场景音效重设计

Cinematic Sound Redesign

重构动作场景音效层级与空间响应，强化叙事感与沉浸感。

Design Focus
  • 枪声层级→按高 / 中 / 低频分为四层（Transient · Body×2 · Tail）叠加素材；主要通过变换 Transient 层制作变体。
  • 脚步声层级→中性脚步声叠加多种材质接触声（Grass / Dirt / Stone / Wood）与衣物摩擦声，配多种变体。
  • 空间混响→山洞内外混响变化，以及过程中的风声变化。
  • 场景氛围→以金属感与爆发冲击的枪声、闷响火声与火星喷溅、变换风声及内外混响，营造危险阴冷的复仇基调。
