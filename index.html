01 · 《冰与铁》—— HP驱动的动态战斗音频系统（Wwise × Unity）

基于HP与伤害参数构建 RTPC 驱动的战斗音频系统，实现命中反馈与音乐情绪的动态演变。

System Implementation：

· RTPC (HP) → 垂直分层音乐 (Vertical Layering)与侧链避让思维（Wwise Meter）：随血量阈值激活音乐分层 Drone 与 Lead，并实现两者间的动态避让。

· RTPC (Damage) → 多阶命中反馈 (Blend Container)：实现命中声音从 Normal 到 Broken、Final Blow 的质感演变。

· State (Game State) → 全局状态切换：基于胜负分晓状态触发全局静默。

Design Goal：

· 建立音频与数值的强耦合，强化玩家对战斗强度及对峙状态（危险或胜利在望）的感知。

02 · 基于RTPC的环境与脚步交互音频系统（Wwise × Unity）

构建基于地面材质与空间距离的交互音频系统，实现脚步声与环境音的动态变化。

System Implementation：

· Switch Group → 物理材质反馈：配置四种地面材质（Wood, Grass, Water, Stone），每种包含 5 个随机变体，消除重复感。

· RTPC（Distance）→ 空间边界预警：建立基于玩家与区域边界距离的实时参数映射，通过脚步声 Pitch 的线性实时偏移，提供直观的距离变化听觉反馈。

Spatial Ambience → 差异化声学环境：根据区域材质同步切换环境底噪（Bed），利用鸟鸣、风声、地矿回响、水声强化不同区域的声学指纹。

Design Goal：

· 通过音频反馈强化玩家对空间、物理边界的感知，利用脚步声音高变化提供直观感性的距离提示，提升交互的功能性。

03 · 混合合成 ——《鸣潮》绯雪技能音效重设计（Hybrid Synthesis Showcase）

本 Demo 为全板内 (In-The-Box) 声音重设计。部分核心元素（Whoosh / Ice Crack / Pulse Wave / Burst / Eject等）通过 Ableton Live 内置工具及外部合成器自制调制生成，部分辅以预设进行材质与超低音铺垫。重点展示了如何利用多参数自动化 (Automation)与滤波扫频 (Filter Sweep)，来塑造冰系技能在蓄力、弹射以及爆发阶段的声压感与物理冲击力。

素材来源： 视频画面源自 IGN 官方演示片段。

制作工具： Ableton Live (Built-in Devices) / Vital / Surge XT / OB-Xd.

Design Focus：

· 混合合成材质： 将 Operator / Wavetable / Vital / Surge XT 自制合成与预置相融合，构建质感统一的、结晶感强烈的冰雪攻击特征以及具有冲击力的技能效果。

· 空间与动态平衡： 利用Glue Compressor、Saturator、Reverb 粘合不同的元素层，统一空间感，同时提升子编组饱和度，实现在提升目标整体响度的同时，不掩蔽高频细节。

04 · 中世纪冷兵器战斧音效设计（Medieval Battle Axe Sound Design）

以《For Honor》音效审美为基准，解构战斧音效音频层级，设计冷兵器物理动态。

Design Focus：

· 材质一致性：通过共享金属基底与谐音层，确保不同动作间音色的连贯；通过共享混响层统一空间感。

· 受击反馈层次：通过多层素材叠加，构建重金属兵器撞击的多样打击感。

· 动态与动能模拟：利用自动化控制与包络修剪，模拟战斧挥动时的动能变化与速度感。

05 · 动作场景音效重设计（Sound Redesign）

重构动作场景音效层级与空间响应，强化叙事感与沉浸感。

Design Focus：

· 枪声层级：按高、中、低频表现分为四层 —— Transient、Body（两层）、Tail， 叠加素材；主要通过变换 Transient 层制作变体。

· 空间混响：山洞内与外部混响变化，以及在这个过程中的风声变化。

· 脚步声层级：中性脚步声叠加多种材质接触声（Grass、Dirt、Stone、Wood等）与衣服摩擦声，配以多种变体。

· 场景氛围：主要通过强调金属感和爆发冲击力的枪声、闷响的火声与火星喷溅、变换的风声与山洞内外不同的混响状态，营造危险阴冷的复仇基调。
